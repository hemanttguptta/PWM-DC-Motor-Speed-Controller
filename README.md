# PWM-Based DC Motor Speed Controller

## Objective
Design and simulate a PWM-based DC motor speed controller using LTspice.

## Components
- 12 V DC Supply
- PWM Voltage Source
- NMOS
- Resistor
- Inductor
- Flyback Diode

## Simulation
Transient analysis (.tran 50m)

## Results
- 25% Duty Cycle
- 50% Duty Cycle
- 75% Duty Cycle
- 100% Duty Cycle

## Conclusion
Motor speed increases with PWM duty cycle because the average voltage applied to the motor increases.
